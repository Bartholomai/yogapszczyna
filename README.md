It is wordpress theme made with love for Yoga.

#### Install dependency
```npm i```

## Build

#### build css file
```npm run build:css```

#### build images file
```npm run build:images```

#### build js file
```npm run build:js```

## Watch

#### run sass watcher
```npm run watch:scss```

#### run images watcher
```npm run watch:images```

#### run js watcher
```npm run watch:js```

## Let's start

#### run all watcher
```npm run start```
